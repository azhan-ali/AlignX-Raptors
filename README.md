## 🔩 The Engineering That Makes It Work

> We didn't just prompt a small model. We **engineered it** to behave like a large one.

| Guard | What It Does |
|---|---|
| 🔐 **Pydantic Validation** | Every single LLM output is clamped against strict type-safe schemas. No hallucination survives. |
| 🔁 **3-Attempt Retry Logic** | If the model breaks JSON, the system auto-simplifies the prompt and retries — up to 3 times. |
| 🛡️ **Deterministic Fallbacks** | If all retries fail, regex + static logic takes over. **The app never crashes. Ever.** |
| 🚫 **Prompt Injection Shield** | All user input is sanitized before it reaches the LLM layer. |
| 🐢 **Rate Limiting** | Built-in pipeline protection against spam and abuse. |

## 🚀 Run It Yourself — Judge's 5-Minute Setup

> **The entire 6-step AI pipeline, running locally on your machine.**

### Prerequisites
- Python 3.10+  |  Node.js 18+  |  [Ollama](https://ollama.ai) installed

### Step 1 — Pull Model & Start AI Backend

```bash
# 1. Pull the tiny-but-mighty 500M model (~300MB)
ollama pull qwen2.5:0.5b
ollama serve

# 2. Clone and set up backend
git clone https://github.com/azhan-ali/AlignX-raptors.git
cd AlignX-raptors/backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

### Step 2 — Launch the Frontend

```bash
cd ../frontend
npm install
npm run dev
```

> 🌐 Open **[http://localhost:3000](http://localhost:3000)** — frontend auto-connects to your local AI backend.
