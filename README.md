## рҹ”© Hardcore Engineering Under the Hood

> **We didn't just prompt a small model. We engineered it to behave like a large one.**

| Guard | What It Does |
|---|---|
| **Pydantic Validation** | Every LLM output is parsed and clamped against strict type-safe schemas. No hallucination survives. |
| **3-Attempt Retry Logic** | If the model breaks JSON structure, the system automatically simplifies the prompt and retries вҖ” up to 3 times. |
| **Deterministic Fallbacks** | If all retries fail, regex + static logic takes over. **The app never crashes.** Ever. |
| **Rate Limiting** | Built-in spam protection for the pipeline. |
| **Prompt Injection Shield** | All user input is sanitized before reaching the LLM layer. |

## рҹҡҖ Run It Yourself вҖ” Judge's Quick Start

> **Get the full AI pipeline running locally in under 5 minutes.**

### Prerequisites
- Python 3.10+
- Node.js 18+
- [Ollama](https://ollama.ai) installed

### Step 1 вҖ” Pull the Model & Start the AI Backend

```bash
# Pull the tiny but mighty 500M model (~300MB download)
ollama pull qwen2.5:0.5b
ollama serve

# Clone the repo
git clone https://github.com/azhan-ali/AlignX-raptors.git
cd AlignX-raptors

# Install backend dependencies & start the server
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

### Step 2 вҖ” Launch the Frontend

```bash
cd frontend
npm install
npm run dev
```
